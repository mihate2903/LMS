**Title: Quản lý Danh mục (Category Management)**

**Description:**
Với tư cách là Quản trị viên, tôi muốn Thêm, Sửa, Xóa danh mục khóa học, để phân loại sản phẩm trên hệ thống.

**Acceptance Criteria:**
1. Cung cấp giao diện để Admin thực hiện các thao tác Thêm, Sửa, Xóa đối với Danh mục (Ví dụ: IT, Thiết kế, Ngoại ngữ).
2. Dữ liệu thay đổi phải được lưu trữ chính xác vào bảng Categories.
3. Các API quản lý danh mục phải được bảo mật và yêu cầu JWT Token của Admin.

**Priority:** Medium
**Effort Estimate:** Low
